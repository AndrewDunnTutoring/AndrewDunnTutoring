Understanding the Link Between Cartesian and Polar Coordinates

This demonstration illustrates the connection between Cartesian and polar coordinate systems.

On the left, the Cartesian graph displays the function \( y = \sin(2x) \). Here, \( x \) is plotted along the horizontal axis, and \( y \) is plotted along the vertical axis. As \( x \) varies, the function creates a wave pattern that oscillates between positive and negative values, effectively illustrating how the amplitude of the function changes with \( x \).

On the right, we visualize the polar function \( r = \sin(2\theta) \). In polar coordinates, \( r \) represents the radial distance from the origin, and \( \theta \) denotes the angle from the positive x-axis. The function \( r = \sin(2\theta) \) produces a rose curve with 4 petals. As \( \theta \) varies, the curve rotates around the origin. The distance from the origin \( r \) changes as \( \theta \) increases, creating the characteristic petal shapes.

The animation will show how the Cartesian curve is mapped into the polar coordinate system. Specifically, as the red dot moves along the Cartesian graph, the corresponding points on the polar graph will be plotted. This process visually demonstrates how each point on the Cartesian curve rotates and transforms into the polar curve, illustrating the connection between the two representations.
